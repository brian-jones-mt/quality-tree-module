### **Scenario 6: IoT Smart Home System**

1.  **Latency:** Commands to smart devices (lights, thermostat) must execute within 200ms.
2.  **Scalability:** Support up to 100 connected devices per household.
3.  **Reliability:** Core home automation routines (e.g., scheduling lights) must always function.
4.  **Security:** Protect the home network from external intrusions and unauthorized device access.
5.  **Interoperability:** Compatible with devices from multiple manufacturers (e.g., Zigbee, Z-Wave, Wi-Fi).
6.  **Privacy:** Ensure all collected sensor data remains private to the homeowner.
7.  **Ease of Setup:** New devices can be added and configured in less than 5 minutes.
8.  **Fault Tolerance:** Continue basic functions even if internet connectivity is lost.
9.  **Battery Life:** Optimize battery usage for wireless sensors to last over 2 years.
10. **User Experience:** Provide a unified and intuitive mobile app interface for all devices.
11. **Voice Control:** Integrate seamlessly with popular voice assistants (e.g., Alexa, Google Assistant).
12. **Firmware Updates:** Allow for secure and over-the-air firmware updates for all devices.
